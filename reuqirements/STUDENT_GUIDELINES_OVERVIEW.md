# MindBridge-RAG Student Project Guidelines

See the PDF file `MindBridge_RAG_Student_Project_Guidelines.pdf` for the complete student-facing project manual.

Core deliverables per group:

1. 1_sources.csv
2. 2_corpus_chunks.csv
3. 3_benchmark_questions.csv
4. 4_ideal_answers.csv
5. 5_risk_labels.csv
6. 6_model_responses.csv
7. 7_human_evaluation.csv
8. group_report.md
9. presentation slides

Main rule: students manually create the corpus and evaluation files; the integration team creates embeddings and the final RAG index.
